# 酿造

## 一般酿造步骤

<video muted="" autoplay="" loop=""> <source src="../../images/cauldron-boiling.Dy3dSJCm.webm"> </video>

- [x] 合成一个炼药锅并灌上水
- [x] 在炼药锅下方放置一下任意一个加热源：
    * [ ] （灵魂）火
	* [ ] 岩浆（块）
	* [ ] （灵魂）篝火
- [x] 右键炼药锅放入原材料
- [x] 使用<b>时钟</b>可以查看炼药锅中的东西已经酿造了多久
- [x] 酿造足够的时间后，用<b> 3 </b>个玻璃瓶完全取出酿制品

对于一般的酿造品，到这里就酿造成功了。但是也有一些还需要[陈酿](aging.md)或者[蒸馏](distilling.md)。
