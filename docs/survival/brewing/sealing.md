# 密封

密封桌使用 2 个玻璃瓶和 4 个木板合成

!!! warning "警告"
	
	酿造品密封后将无法再继续陈酿或者蒸馏，且不再显示详细信息。
	
	相似配方的酿造品密封后将成为同一个酿造品。
